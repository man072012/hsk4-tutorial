# 📚 شرح اختبار HSK4 المحاكي رقم 6

دليل تعليمي شامل بالعربية لاختبار HSK4 الصيني - تفكيك كل كلمة وقاعدة بأسلوب الفهم بدون حفظ.

## ✨ الميزات

- 🎧 **45 سؤال استماع** مع الصوت الأصلي
- 📖 **40 سؤال قراءة** مع الشرح الكامل
- ✏️ **15 سؤال كتابة** مع الجمل النموذجية
- 🎨 **54 بطاقة ذاكرة بصرية** برسومات SVG
- 🔊 **3 محركات نطق صيني** (Youdao + Google + Web Speech)
- 🎤 **تسجيل صوتك** ومقارنته بالنطق الصحيح
- 🔑 **تظليل الكلمات المفتاحية** في كل سؤال
- 👁️ **زر إخفاء Pinyin** لاختبار الحفظ
- 📐 **القواعد المفتاحية** كاملة
- 📍 **روابط للأسئلة** من كل بطاقة

## 🚀 الاستخدام

افتح `index.html` في المتصفح (يفضّل Chrome أو Edge).

[**جرّب الدليل مباشرة هنا** ↗](https://YOUR-USERNAME.github.io/hsk4-tutorial/)
