# 📚 HSK4 - دليل الاختبارات المحاكية بالعربية

دليل تعليمي شامل بالعربية لاختبارات HSK4 الصينية المحاكية - تفكيك كل كلمة وقاعدة بأسلوب الفهم بدون حفظ.

## 🌐 رابط الدليل

[**جرّب الدليل مباشرة هنا** ↗](https://man072012.github.io/hsk4-tutorial/)

اختر الاختبار من صفحة الهبوط:
- **Mock Test Paper 6** - الاختبار الأول الكامل
- **Mock Test Paper 9** - الاختبار الثاني مع تحسينات

## ✨ الميزات

- 🎧 **45 سؤال استماع** مع الصوت الأصلي MP3
- 📖 **40 سؤال قراءة** مع الشرح الكامل
- ✏️ **15 سؤال كتابة** مع الجمل النموذجية
- 🎨 **54-57 بطاقة ذاكرة بصرية** برسومات SVG
- 🔊 **4 محركات نطق صيني** (iFlytek + Youdao + Google + Web Speech)
- 🎤 **تسجيل صوتك** ومقارنته بالنطق الصحيح
- 🔑 **تظليل الكلمات المفتاحية** في كل سؤال
- 👁️ **زر إخفاء Pinyin** لاختبار الحفظ
- 📐 **القواعد المفتاحية** كاملة (10-20 قاعدة)
- ⚡ **تحميل أصوات مسبق** للتشغيل الفوري (Mock 9)

## 📁 الملفات

- `index.html` - صفحة الهبوط (اختر بين Mock 6 و Mock 9)
- `mock6.html` - دليل الاختبار رقم 6
- `mock9.html` - دليل الاختبار رقم 9

## 🔧 التقنيات

- HTML5 + CSS3 + JavaScript (vanilla)
- Web Speech API للنطق الاحتياطي
- iFlytek WebSocket TTS مع HMAC-SHA256
- Youdao TTS / Google Translate TTS
- MediaRecorder API للتسجيل
- LocalStorage لحفظ الإعدادات
- SVG inline للبطاقات البصرية
