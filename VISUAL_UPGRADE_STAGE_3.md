# HSK4 Visual Upgrade — Stage 3

## نطاق الجزء الثالث

هذا الجزء يركز على `mock6.html` فقط، مبني على نسخة الجزء الثاني، حتى تبقى تعديلات `index.html` و `mock9.html` موجودة.

## ماذا تغير؟

- إضافة هوية داخلية جديدة لصفحة الاختبار رقم 6.
- شريط علوي احترافي متوافق مع صفحة البداية وصفحة Mock 9.
- إعادة تصميم Hero الخاص بالاختبار رقم 6.
- تحسين تبويبات التنقل الداخلية.
- إعادة تصميم عميق لبطاقات الأسئلة، وليس فقط تغيير ألوان خارجية.
- تحسين بطاقات الصوت، النص الصيني، الترجمة، المفردات، الشرح، سبب الإجابة، والذاكرة.
- تحسين بطاقات الذاكرة البصرية والقواعد في قسم المرجع الشامل.
- تحسين الأزرار العائمة، مؤشر التقدم، وحالة الوضع الليلي.
- إضافة مؤثرات انتقالية وحركة خفيفة بدون تغيير المحتوى الأكاديمي.

## ما لم يتغير

- لم يتم تغيير الأسئلة.
- لم يتم تغيير الإجابات.
- لم يتم تغيير الشروحات الأكاديمية.
- لم يتم تغيير روابط الصوت.
- لم يتم حذف أي محتوى تعليمي.
