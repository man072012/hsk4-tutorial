# HSK4 Visual Upgrade — Stage 5 Release Hardening

هذه المرحلة ليست تغييراً للمحتوى الأكاديمي، بل تجهيز إصدار الإنتاج حتى يكون الموقع صالحاً للرفع على GitHub Pages بصورة أكثر نضجاً.

## ما أضيف في الجزء 5

- إضافة طبقة نهائية للإنتاج:
  - `assets/hsk4-release-hardening.css`
  - `assets/hsk4-release-hardening.js`
- إضافة قابلية التثبيت كتطبيق ويب خفيف PWA:
  - `site.webmanifest`
  - `sw.js`
  - `assets/icons/icon.svg`
  - `assets/icons/maskable-icon.svg`
- إضافة أدوات صفحة عائمة:
  - الرجوع لأعلى الصفحة.
  - الطباعة / الحفظ PDF.
  - نسخ رابط الصفحة.
- تحسين تجربة الجوال والطباعة.
- تحسين التحميل الكسول للصور.
- إضافة تمييز للرابط الحالي في التنقل.
- إضافة اختصارات لوحة مفاتيح بسيطة:
  - `Alt + T` للرجوع لأعلى الصفحة.
  - `Alt + P` للطباعة.
- تجهيز ملفات نشر آمنة على GitHub بدون وضع أي Token داخل الملفات.

## ما لم يتم تغييره

- لم يتم تغيير الأسئلة.
- لم يتم تغيير الإجابات.
- لم يتم تغيير الشروحات.
- لم يتم تغيير ملفات البيانات.
- لم يتم تغيير مسارات صفحات GitHub Pages.

## ملاحظة مهمة عن Service Worker

بعد الرفع على GitHub Pages قد يحتاج المتصفح إلى تحديث قسري مرة واحدة حتى يلتقط النسخة الجديدة:

- Windows: `Ctrl + F5`
- أو افتح DevTools ثم Application ثم Service Workers ثم Update/Unregister عند الحاجة.

لإيقاف تسجيل Service Worker مؤقتاً أثناء التجربة أضف إلى الرابط:

`?no-sw`
