# HSK4 Visual Upgrade — Stage 4 Final

هذه المرحلة هي مرحلة التوحيد والمراجعة النهائية، مبنية على مراحل التصميم السابقة.

## ما تم إنجازه

- توحيد الهوية البصرية بين:
  - `index.html`
  - `mock6.html`
  - `mock9.html`
- إضافة طبقة نهائية مشتركة:
  - `assets/hsk4-final-polish.css`
  - `assets/hsk4-final-polish.js`
- تحسين تجربة الجوال:
  - زر قائمة موحّد للشاشات الصغيرة.
  - ترتيب أفضل للبطاقات والقوائم الطويلة.
  - تقليل الازدحام في واجهات الاختبار.
- تحسين الوصول والقراءة:
  - حالات تركيز واضحة للوحة المفاتيح.
  - دعم `prefers-reduced-motion` لمن يفضّل تقليل الحركة.
  - تحسين الطباعة/الحفظ PDF بإخفاء عناصر التحكم غير الضرورية.
- تحسين الأداء البصري:
  - استخدام `content-visibility` للعناصر الطويلة عند الدعم.
  - انتقالات ناعمة دون لمس المحتوى الأكاديمي.
- مراجعة الروابط والملفات الأساسية.

## ما لم يتم تغييره

- لم يتم تغيير الأسئلة.
- لم يتم تغيير الإجابات.
- لم يتم تغيير الشروحات التعليمية.
- لم يتم تغيير ملفات البيانات.
- لم يتم تغيير بنية روابط GitHub Pages الأساسية.

## طريقة الرفع على GitHub Pages

1. افتح مستودع `hsk4-tutorial` في GitHub.
2. ارفع محتويات مجلد `hsk4-project-full` كما هي إلى جذر المستودع.
3. اعمل Commit باسم مثل:
   `Final visual redesign for HSK4 tutorial`
4. انتظر تحديث GitHub Pages.
5. افتح الرابط:
   `https://man072012.github.io/hsk4-tutorial/`

لا تضع GitHub token داخل الملفات أو المحادثة أو GitHub. إذا تم كشف token سابقاً، ألغِه من إعدادات GitHub وأنشئ token جديداً عند الحاجة.
